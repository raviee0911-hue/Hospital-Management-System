This Python file is a part of the Hospital Management System project and is mainly responsible for managing patient records and billing operations. It uses database connectivity, SQL queries, and object-oriented programming concepts to handle hospital data efficiently. 

### Key Features:

* Creates and manages the **patients database table**
* Adds new patient records with:

  * Patient details
  * Disease validation
  * Doctor suggestion system
  * Bed allocation option
* Views patient details using:

  * Patient ID
  * Phone number
* Updates patient information
* Deletes patient records from the database
* Generates patient bills automatically with:

  * Doctor fees
  * Hospital charges
  * Bed charges
  * Total amount calculation
* Saves the generated bill as a **text file** in the Downloads folder

### Technologies Used:

* **Python**
* **SQL Database Connectivity**
* **Regular Expressions (Regex)**
* **File Handling**
* **Object-Oriented Programming (OOP)**

This module improves hospital workflow by ensuring accurate patient data management, billing automation, and faster record handling.
